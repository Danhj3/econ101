# Instrucciones editoriales

## Tono de voz
Formal, académico, claro, sin tono comercial ni espiritual.

## Destacar
Investigación, educación intercultural, políticas públicas educativas, publicaciones, docencia y conferencias.

## Evitar
Autoayuda, esoterismo, marketing de redes, exageraciones de autoridad.

## Guía para entradas de blog
- Problema educativo.
- Marco conceptual.
- Evidencia o experiencia.
- Implicaciones para política pública.
- Referencias.

## Criterios para publicar conferencias
Fecha, institución, lugar, tema, enlace verificable o “pendiente de confirmar”.

## Criterios para registrar publicaciones
Título, año, tipo, resumen, estado (publicado/en preparación), enlace.

## Reglas de citas
Usar formato consistente (APA sugerido), sin inventar DOI/índices.
