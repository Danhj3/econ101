# Sitio académico de Solia Centeno

## Objetivo
Presentar una web académica sobria para posicionar a Solia María Centeno Castro de Baglivo en investigación educativa y políticas públicas.

## Estructura
- `index.html`, `investigacion.html`, `publicaciones.html`, `docencia.html`, `conferencias.html`, `cv.html`, `contacto.html`, `privacidad.html`
- `assets/css/style.css`, `assets/js/main.js`
- `assets/img/*.svg` placeholders
- `assets/docs/cv-solia-centeno.pdf` placeholder

## Edición de contenidos
1. Editar textos en cada HTML.
2. Mantener menú idéntico en todas las páginas.
3. Reemplazar placeholders `[pendiente...]` cuando exista confirmación.

## Publicación
- Subir archivos por FTP o despliegue estático (Netlify, Vercel static, cPanel).
- Configurar dominio `soliacenteno.com` y SSL.

## Datos faltantes
Correo, ORCID, Google Scholar, ResearchGate, LinkedIn, enlaces de publicaciones, agenda verificable de conferencias.

## Reemplazo de imágenes
Sustituir SVG en `assets/img/` por fotografías optimizadas y mantener `alt` descriptivos.

## Actualización de CV/publicaciones
Actualizar `cv.html`, `publicaciones.html` y PDF trimestralmente.
